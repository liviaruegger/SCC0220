Este teste foi realizado no GDB Online para evitar as otimizações de branch prediction do meu computador
(que melhoram o significativamente o desempenho da ordenação no pior caso).

Foram testados três tipos de vetores, todos com tamanho 1000:
Tipo 0 - Vetor com valores aleatórios.
Tipo 1 - Vetor ordenado.
Tipo 2 - Vetor inversamente ordenado.
